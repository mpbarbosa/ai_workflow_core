# Shell Script Integration Example

This example shows how to integrate `ai_workflow_core` into a shell script project.

## Quick Start

```bash
# Copy this example to get started
cp -r .workflow_core/examples/shell my_shell_project
cd my_shell_project

# Initialize git and add submodule
git init
git submodule add https://github.com/mpbarbosa/ai_workflow_core.git .workflow_core

# Customize configuration
# Edit .workflow-config.yaml with your project details
```

See full setup instructions and best practices in the complete README.

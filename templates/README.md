# templates/

This directory contains **reusable template files and checklists** that projects can copy and adapt.

## Subdirectories

| Directory | Contents |
|-----------|----------|
| [`debugging/`](debugging/README.md) | Four Markdown debugging checklists for async flows, browser APIs, data structures, and observer patterns. |

## Usage

Copy the relevant file(s) into your project and customise as needed. Templates are written to be language- and framework-agnostic unless noted otherwise.
